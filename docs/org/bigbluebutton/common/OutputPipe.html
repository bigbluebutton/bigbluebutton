public function write(message:IPipeMessage):Boolean

To send a message, add a header with SRC and TO fields as a convention. The Router will look at the TO field and routes the message to the correct inputpipe for a module. e.g. // create a message var messageToSend:IPipeMessage = new Message( Message.NORMAL, { SRC:'MAININPUT', TO: 'LOGGERINPUT' }, new XML(), Message.PRIORITY_HIGH );
