BigBlueButton 0.81-dev Overview

Welcome

BigBlueButton is an open source web conferencing system for on-line learning.

It supports sharing of presentations (PDF and any document readable by OpenOffice), webcams, chat, audio, and desktop with remote viewers. It can record and playback sessions for later viewing (presentation, webcams, chat, and audio). The client runs on Mac, Unix, and PC computers. The project is supported by a community of developers that care about good design and a streamlined user experience.
